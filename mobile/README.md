# Marketplacex Flutter App

Marketplacex is a clean Flutter marketplace app with Firebase email/password authentication, a searchable product grid, product details, cart management, and checkout.

## Run the app

```bash
flutter pub get
flutter run
```

On Windows, Firebase plugins require symlink support. Enable Developer Mode in Windows Settings if Flutter prints a symlink warning.

## Firebase setup

This project includes `lib/firebase_options.dart` with placeholder values so the app structure is complete. Replace it with real Firebase configuration:

```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

Then enable Email/Password authentication in the Firebase Console.

## File map

- `pubspec.yaml`: Flutter project metadata and dependencies, including `firebase_core`, `firebase_auth`, `provider`, and `intl`.
- `lib/main.dart`: App entrypoint. Initializes Firebase and registers app-level providers.
- `lib/firebase_options.dart`: Firebase platform configuration. Replace placeholders with real FlutterFire output.
- `lib/app/marketplacex_app.dart`: Root `MaterialApp` and route entry through the auth gate.
- `lib/theme/app_theme.dart`: Shared Material 3 theme for the modern UI.
- `lib/models/product.dart`: Product data model.
- `lib/models/cart_item.dart`: Cart line item model.
- `lib/data/sample_products.dart`: Demo product catalog used by the home grid.
- `lib/providers/product_provider.dart`: Product search/filter state.
- `lib/providers/cart_provider.dart`: Add, remove, quantity, subtotal, shipping, and total cart state.
- `lib/services/auth_service.dart`: Firebase Authentication wrapper for login, signup, auth stream, and sign out.
- `lib/utils/currency.dart`: Shared currency formatter.
- `lib/screens/auth/auth_gate.dart`: Switches between auth screens and the home page based on Firebase login state.
- `lib/screens/auth/login_page.dart`: Email/password login screen.
- `lib/screens/auth/signup_page.dart`: Account creation screen.
- `lib/screens/home/home_page.dart`: Search bar, product grid, cart shortcut, and sign-out drawer.
- `lib/screens/product/product_details_page.dart`: Product details and add-to-cart action.
- `lib/screens/cart/cart_page.dart`: Cart item list, quantity controls, subtotal, and checkout navigation.
- `lib/screens/checkout/checkout_page.dart`: Shipping form, order summary, and place-order flow.
- `lib/widgets/product_card.dart`: Reusable product grid card.
- `test/widget_test.dart`: Smoke test for the marketplace home UI.
- `android/`, `ios/`, `web/`, `windows/`, `macos/`, `linux/`: Native Flutter platform scaffolding generated by `flutter create`.
